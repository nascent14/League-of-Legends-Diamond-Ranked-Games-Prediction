# League-of-Legends-Diamond-Ranked-Games-Prediction
This project was conducted as part of the Artificial Intelligence and Machine Learning course (CSE 3202) and focused on predicting the outcome of League of Legends diamond-ranked matches using gameplay statistics from the first ten minutes. The dataset, sourced from Kaggle and originally collected from Riot Games, consisted of approximately 10,000 high-ELO ranked matches with detailed team-based performance metrics. The project involved data preprocessing and feature selection, exploratory data analysis, and training multiple classification models including Logistic Regression, Gaussian Naive Bayes, XGBoost, and Random Forest using both 70/30 and 80/20 train-test splits. Model performance was evaluated using accuracy, precision, recall, F1-score, and confusion matrices, with Logistic Regression achieving the highest predictive accuracy.
